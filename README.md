# 🚗 Toy Cybertruck – Voice-Controlled Robotic Car

A voice-controlled robotic car built using **Arduino Uno**, **VC-02 Offline Voice Recognition Module**, and **Bluetooth communication (HC-05/HC-06)**. This project demonstrates how embedded systems, offline speech recognition, and wireless communication can be integrated to create an interactive robotic platform.

---

## 📌 Overview

Toy Cybertruck is a custom-designed robotic car that responds to offline voice commands such as **Forward**, **Backward**, **Left**, **Right**, and **Stop**. The system uses the VC-02 voice recognition module to recognize predefined voice commands without requiring an internet connection. The recognized commands are transmitted via Bluetooth to the Arduino Uno, which controls the motors through an L293D motor driver.

This project highlights practical implementation of embedded systems, IoT concepts, Bluetooth communication, motor control, and rapid hardware prototyping.

## ⚙️ Working

1. The user gives a voice command.
2. The VC-02 module recognizes the command offline.
3. The recognized command is sent through the HC-05 Bluetooth module.
4. The HC-06 Bluetooth module connected to the Arduino Uno receives the command.
5. Arduino processes the received command.
6. The L293D Motor Driver drives the appropriate motors.
7. The robotic car performs the requested movement.
